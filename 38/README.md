# Day 38 – Smart Password System with LCD & EEPROM

## Overview

This project implements a secure password-based access system using a keypad for input, LCD for display, and EEPROM for permanent storage.

---

## Components Used

* Arduino Uno
* 4x4 Keypad
* 16x2 I2C LCD
* EEPROM (internal)
* LED / Buzzer

---

## Working Principle

* User enters password via keypad
* LCD displays masked input
* System compares with stored password
* Access is granted or denied
* Password can be updated and stored permanently

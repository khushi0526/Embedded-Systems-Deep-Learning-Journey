# Day 31 – Password-Based Access System (Keypad)

## Overview

This project uses a 4x4 keypad to implement a password-based access system. The user enters a password, and the system validates it to grant or deny access.

---

## Components Used

* 4x4 Keypad
* Arduino Uno
* LED / Buzzer

---

## Working Principle

* User presses keys
* Input is stored
* Compared with predefined password
* Access granted or denied

---

## Simulation Link

https://wokwi.com/projects/461364118402926593

---

## System Logic

* Correct password → LED ON
* Wrong password → buzzer alert

---

## Learning Outcomes

* Matrix input system
* String handling
* Security logic
* User interaction

---

## Conclusion

This project demonstrates a simple yet powerful authentication system used in real-world applications.

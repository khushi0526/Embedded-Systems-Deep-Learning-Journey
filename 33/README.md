# Day 33 – Real-Time Clock (RTC) System

## Overview

This project uses an RTC module (DS1307/DS3231) to track real-world time and perform actions based on it.

---

## Components Used

* RTC Module (DS1307 / DS3231)
* Arduino Uno
* LED / Buzzer

---

## Working Principle

* RTC keeps track of real time
* Arduino reads time via I2C
* Actions are triggered based on time

---

## System Logic

* Example:

  * 6 PM → LED ON
  * 6 AM → LED OFF

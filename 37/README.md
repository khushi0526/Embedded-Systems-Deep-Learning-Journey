# Day 37 – LCD Menu-Based Control System

## Overview

This project implements a menu-driven user interface using an LCD display. The system allows users to navigate through different screens, view sensor data, and control system modes.

---

## Components Used

* Arduino Uno
* 16x2 I2C LCD
* Push Button
* LDR Sensor
* LED

---

## Working Principle

* Button press changes menu screen
* LCD displays system information
* User can view sensor data and system mode
* LED operates based on selected mode

---

## Menu Screens

1. Temperature / Light Info
2. LED Status
3. Mode (AUTO / MANUAL)
4. Instructions

---

## Learning Outcomes

* UI design in embedded systems
* Menu-based navigation
* State management
* Human-machine interaction

---

## Conclusion

This project demonstrates how embedded systems can provide user-friendly interfaces similar to real-world devices.

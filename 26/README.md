# Day 26 – Smart Room System with LCD Interface

## Overview

This project enhances the smart room system by adding a 16x2 LCD display. The LCD provides real-time feedback about temperature, light status, and system mode.

---

## Components Used

* LDR
* DHT11
* Servo Motor
* LED
* Buzzer
* Push Button
* 16x2 LCD Display
* Arduino Uno

---

## Working Principle

* Sensors collect data
* System processes logic
* LCD displays real-time information

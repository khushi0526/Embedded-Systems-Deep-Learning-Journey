# Day 32 – EEPROM Data Storage System

## Overview

This project uses EEPROM to store and retrieve data permanently. Even after power OFF or reset, the stored data remains available.

---

## Components Used

* Arduino Uno (Internal EEPROM)
* Keypad
* LED / Buzzer

---

## Working Principle

* User enters password
* Password is stored in EEPROM
* On restart, system reads saved password
* No need to reprogram

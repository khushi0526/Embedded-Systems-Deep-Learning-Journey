# Day 30 – Object Detection using IR Sensor

## Overview

This project uses an IR sensor to detect objects. When an object is detected, an LED and buzzer are triggered to alert the user.

---

## Components Used

* IR Sensor Module
* LED
* Buzzer
* Arduino Uno

---

## Working Principle

* IR sensor emits infrared light
* When object is near → reflection detected
* Sensor outputs LOW/HIGH signal
* Arduino triggers alert

## System Logic

* Object detected → LED ON + buzzer beep
* No object → system OFF

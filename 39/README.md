# Day 39 – Multi-Sensor Smart Monitoring System

## Overview

This project integrates multiple sensors into a single system to monitor environmental conditions and provide alerts. It simulates real-world monitoring systems used in industries and smart environments.

---

## Components Used

* Arduino Uno
* DHT22 Temperature Sensor
* LDR Sensor
* 16x2 I2C LCD
* LED
* Buzzer

---

## Working Principle

* Sensors continuously collect data
* Arduino processes readings
* LCD displays real-time values
* Alerts triggered when thresholds exceed

---

## System Logic

* High temperature → buzzer alert
* Low light → LED ON
* LCD shows sensor data

---

## Learning Outcomes

* Multi-sensor integration
* Threshold-based control
* Embedded monitoring system design
* Real-time data display

---

## Conclusion

This project demonstrates how embedded systems can monitor environmental parameters and respond intelligently, forming the basis for IoT and industrial systems.

# Day 28 – Smart Environment Control System

## Overview

This project represents a complete embedded system integrating sensing, control, user interface, and manual override. It simulates a smart environment controller used in automation systems.

---

## Features

* Multi-sensor input (Temperature + Light)
* LCD menu interface
* Automatic control system
* Manual override mode
* Servo-based vent control
* Intelligent alert system (buzzer tones)

---

## Modes

### AUTO MODE

* LDR controls lighting
* Temperature controls vent
* High temperature triggers alert

### MANUAL MODE

* User controls LED and vent
* Alerts disabled

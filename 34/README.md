# Day 34 – Serial Communication Control System

## Overview

This project demonstrates communication between a computer and Arduino using Serial Monitor. The user sends commands to control devices in real time.

---

## Components Used

* Arduino Uno
* LED
* Servo Motor

---

## Working Principle

* User sends command via Serial Monitor
* Arduino reads input
* Executes corresponding action

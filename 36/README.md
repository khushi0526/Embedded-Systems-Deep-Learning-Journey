# Day 36 – Smart Command + Sensor System

## Overview

This project combines sensor-based automation with user command control. It simulates a smart system that can operate both automatically and manually.

---

## Components Used

* Arduino Uno
* LDR Sensor
* LED
* Serial Monitor (acts as control interface)

---

## Working Principle

* System reads LDR value
* User can switch between AUTO and MANUAL modes
* In AUTO → LED depends on light
* In MANUAL → LED controlled by user commands

---

## Commands

* 'A' → AUTO Mode
* 'M' → MANUAL Mode
* '1' → LED ON (Manual)
* '0' → LED OFF (Manual)

---

## Learning Outcomes

* Hybrid system design
* Mode switching logic
* Combining sensor + user control
* Real-world automation thinking

---

## Conclusion

This project demonstrates a smart control system integrating automation and manual override, a key concept in modern embedded systems.

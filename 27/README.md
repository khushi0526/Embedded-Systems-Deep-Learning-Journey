# Day 27 – LCD Menu-Based Smart System

## Overview

This project introduces a menu-driven interface using a 16x2 LCD. Users can navigate through different system parameters using a button, improving interaction and usability.

---

## Components Used

* 16x2 LCD
* Push Button
* DHT11
* LDR
* Servo Motor
* Arduino Uno

---

## Working Principle

* Button press → change screen
* Each screen displays different data
* System continues running in background

---

## Menu Screens

1. Temperature Display
2. Light Status
3. Vent Position
4. Mode Status

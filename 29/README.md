# Day 29 – Distance Measurement System (Ultrasonic Sensor)

## Overview

This project uses an ultrasonic sensor (HC-SR04) to measure distance and display it via Serial Monitor. It simulates applications like obstacle detection and smart parking systems.

---

## Components Used

* HC-SR04 Ultrasonic Sensor
* Arduino Uno
* (Optional: Buzzer/LED for alert)

---

## Working Principle

* Sensor sends ultrasonic wave
* Measures time for echo return
* Calculates distance

# Embedded Systems Learning Journey

This repository documents my continuous learning in embedded systems through simulations and open-source tools. It focuses on understanding sensors, actuators, microcontrollers, and core electronics concepts in a structured and practical way.

Each project reflects my approach to learning—writing code, analyzing behavior, and building system-level understanding rather than just implementation.

The goal is to develop strong fundamentals and problem-solving skills, preparing for advanced work in embedded systems and VLSI.

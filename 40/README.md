# Day 40 – Smart Embedded System 

## Overview

This project integrates multiple embedded system components into a single smart system. It combines authentication, monitoring, control, and user interface features, simulating a real-world intelligent device.

---

## Components Used

* Arduino Uno
* 4x4 Keypad
* 16x2 I2C LCD
* DHT22 Sensor
* LDR Sensor
* EEPROM (internal)
* LED
* Buzzer

---

## Working Principle

* User enters password to access system
* LCD provides interaction and feedback
* Sensors monitor environment conditions
* System performs automatic control
* Alerts triggered for abnormal conditions

## System Modules

2. Monitoring System (DHT22 + LDR)
3. Display System (LCD)
4. Control System (LED + logic)
5. Alert System (Buzzer)
